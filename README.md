# 🚀 Code Platform

A modern **Full Stack Coding Practice Platform** developed using **Next.js, React, Node.js, Express.js, Prisma ORM, PostgreSQL, and TypeScript**.

The platform enables users to practice coding problems, monitor their learning progress, view leaderboards, manage submissions, and earn certificates. It also provides an admin panel for managing coding problems and monitoring user activity.

---

# 📸 Project Overview

Code Platform is designed to simulate an online coding platform where users can:

* Practice programming problems
* Save and manage submissions
* Track coding progress
* View rankings on the leaderboard
* Earn certificates after completing language tracks

Administrators can manage coding problems and monitor user submissions through a secure admin dashboard.
