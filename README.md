# 🚀 Code Platform

A Full Stack Coding Practice Platform built using **Next.js, React, Node.js, Express.js, Prisma ORM, PostgreSQL, and TypeScript**.

This platform helps users practice coding problems, track their progress, compete on leaderboards, and earn certificates after completing language-specific challenges.

---

# ✨ Features

## 👤 User Features

* User Registration & Login
* JWT Authentication
* Solve Coding Problems
* Code Editor (Monaco Editor)
* Run & Save Code
* Dashboard
* Language Progress Tracking
* Leaderboard
* Submission History
* Public User Profile
* Certificates
* Responsive UI

---

## 🛠 Admin Features

* Admin Login
* Add New Problems
* Edit Existing Problems
* Delete Problems
* View User Submissions
* Admin Notifications
* Protected Admin Routes
